## Tech Stack

- **Frontend:** Next.js 16, React 19, Tailwind CSS
- **Backend:** Next.js API Routes, TypeScript
- **Database:** PostgreSQL (NeonDB)
- **Authentication:** Clerk
- **File Uploads:** UploadThing
- **Validation:** Zod
- **AI Framework:** LangChain
- **LLM:** Google Gemini API
- **Deployment:** vercel
